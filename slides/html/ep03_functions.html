functions

what is a function?

anything that operates on an input

x → f(x) → y

to determine an output

for example

the area of a circle is a function of its radius

\[ A = \pi r^{2} \] or \[ f(x) = \pi x^{2} \]

so why do we like them?

functions allow for

brevity

in describing one or many processes

for example

say I have an equation for calcluating compound interest

..

\[ P' = P \left(1 + \frac{r}{n}\right)^{nt} \]

..

where $P$ = principal sum , $P'$ = new sum , $r$ = nominal interest rate , $n$ = compounding frequency , $t$ = time of investment

it would laborious to write this out every time I have some different permutation of inputs

rather I would be better defining it like so

\[ f(P, r, n, t) = P \left(1 + \frac{r}{n}\right)^{nt} \]

..

so that we can just write

\[ P' = f(P, r, n, c) \]

this mathematical abstraction

to encapsulate and execute any arbitrary process at will

applies to programming as well

    """
    iteratively sums numbers from 1 to N
    """
    sum_i = 0

    for i in range(N + 1):
        sum_i += i

    return sum_i

under the hood

**kwargs

works similarly to *args but uses key:value pairing instead


def database_entry(name, **ancilliary):
    """adds a new entry to a database"""
    entry = {}
    entry['name'] = name

    for (key, value) in ancilliary.items():
        entry[key] = value

    return entry
