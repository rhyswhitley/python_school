what is a function?

anything that operates on an input

x → f(x) → y

to determine an output

for example

the area of a circle is a function of its radius

\[ A = \pi r^{2} \] or \[ f(x) = \pi x^{2} \]

mathematically

rather than write out this

\[ \pi 2^{2} = 12.57 \\ \pi 3^{2} = 28.27 \\ \pi 4^{2} = 50.27 \\ ... \]
I can simplify to

\[ f(2) = 12.57 \\ f(3) = 28.27 \\ f(4) = 50.27 \\ ... \]

for example

we could create a function to calculate the area of a circle


def circle_area(radius):
    """
    Calculates the area of a circle
    """

    # define pi (I would usually import this)
    pi = 3.14159265359

    # calculate area
    area = pi*radius**2

    # return value
    return area

we can also call functions from inside other functions


def cylinder_volume(height, radius):
    """
    Calculates the volume of a cylinder
    """
    
    # calculate area
    x_area = circle_area(radius)

    # calculate volume
    volume = x_area*height

    # return value
    return volume
